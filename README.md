# iRacingSlipAngleVisualizer (iSAV)

**iRacing** のテレメトリーデータをリアルタイムで解析し、スリップアングル・ペダル入力・G力などをゲーム画面にオーバーレイ表示するドライビング解析ツールです。

> **iRacing 非公式ツールです。iRacing とは一切関係ありません。**
> 無料・個人利用のみ

## 主な機能

| 機能 | 説明 |
|------|------|
| スリップアングル表示 | 前後左右4輪・CoGのスリップ角をリアルタイム数値表示 |
| スリップベクター図 | タイヤごとのスリップ方向を矢印ベクターで可視化 |
| 荷重移動バー | 4輪それぞれの外側に縦バーで荷重変化を表示（オレンジ=荷重増・ブルー=荷重減） |
| リアルタイムグラフ | スリップ角の時系列グラフ（5/10/30/60秒ウィンドウ切替） |
| ペダル入力グラフ | アクセル（緑）・ブレーキ（赤）・クラッチ（オレンジ）の開度グラフ（各ライン表示切替可） |
| タイヤスリップバランスモニター | フロント/リアのスリップ角バーメーター + NEUTRAL/UNDERSTEER/OVERSTEER リアルタイム判定。時系列グラフ付き |
| FFB Grip Analysis | FFB信号からグリップ使用率・スリップ効率を算出。NORMAL/NEAR LIMIT/OVER LIMITをリアルタイム判定。スナップオーバーステア検知＋フラッシュ警告、時系列グラフ付き |
| G-Force トラクションサークル | 前後左右Gを円グラフで可視化（軌跡付き）。車両タイプ別スケール自動切替 |
| ABS 作動検知 | ブレーキ中の ABS 作動をトラクションサークルのオレンジ点滅で表示 |
| SDK ローデータ | Speed / RPM / Fuel / Gear / 気温 / 路面温度 / Velocity / YawRate などの生テレメトリー |
| セッションリプレイ解析（ANALYSIS） | セッション後ガレージに戻ってリプレイを再生すると過去データを追従表示 |
| IBT ファイル再生 | .ibt 録画の読み込み・再生・逆再生・コマ送り対応 |
| グラフウィンドウ分離 | グラフを独立ウィンドウへ切り出して自由配置 |
| レブリミットビープ | 車種ごとのレブリミットを自動取得・警告ビープ音（閾値調整可） |
| クリックスルー | パネル外エリアはマウスを透過し iRacing を直接操作可能 |
| ウィンドウ設定記憶 | 位置・サイズ・グラフ設定を終了時に自動保存・復元 |

## 動作環境

- Windows 10 / 11（64 ビット）
- iRacing がインストール済みであること（セッションモード使用時）

## インストール

1. [Releases ページ](https://github.com/liquidfinger/iRacing_App_release/releases/latest) から `iRacingSlipAngleVisualizer_Setup_v1.3.0.5.exe` をダウンロード
2. インストールウィザードに従って完了
3. スタートメニュー または デスクトップショートカットから起動

---

## クイックスタート

**セッションモード（iRacing 走行中）：**
1. iRacing を起動してセッションに参加
2. iRacingSlipAngleVisualizer を起動
3. コントロールパネルの **「Show Overlay」** をクリック
4. ゲーム画面にオーバーレイが表示されます

**リプレイ再生モード（iRacing 保存リプレイ再生）：**
1. iRacingSlipAngleVisualizer を起動する
2. iRacing を起動して保存リプレイファイルを開く
3. コントロールパネルの **「Show Overlay」** をクリック
4. iRacing 上でリプレイを再生するとスリップ角・G-Force・ペダル入力などが追従表示される

**IBT リプレイモード：**
1. コントロールパネルの「Data Source」タブを開く
2. 「IBT File」を選択して .ibt ファイルを指定
3. 「Show Overlay」でオーバーレイを表示
4. 再生コントロールで再生・逆再生・コマ送り操作
