# iRacingSlipAngleVisualizer

**iRacing** のテレメトリーデータをリアルタイムで解析し、スリップアングルをわかりやすく視覚表示するオーバーレイアプリです。
ゲーム画面に半透明オーバーレイを重ねて表示するため、走行中のドライビングフィードバックとして活用できます。

> **iRacing 非公式ツールです。iRacing とは一切関係ありません。**
> 無料・個人利用のみ

## 主な機能

| 機能 | 説明 |
|------|------|
| スリップアングル表示 | 前後左右4輪・CoGのスリップ角をリアルタイム数値表示 |
| スリップベクター図 | タイヤごとのスリップ方向を視覚的なベクター図で表示 |
| リアルタイムグラフ | スリップ角の時系列グラフ（5秒ウィンドウ）、グラフ分離ウィンドウ対応 |
| ペダル入力グラフ | アクセル（緑）・ブレーキ（赤）の開度グラフ |
| SDKローデータ | Speed / Fuel / AirTemp / TrackTemp などの生テレメトリー表示 |
| IBT再生 | iRacing の .ibt 録画ファイルを読み込んで解析・再生 |
| 外部車両DB | carspec.json から車両パラメーター（WB・重心・ステア比）を自動取得（42車種） |
| オーバーレイヘッダー | 車両名・Dry/Wet 路面状況をリアルタイム表示 |
| ドラッグ移動 | オーバーレイ内の各パネルを個別に移動・配置カスタマイズ |
| クリックスルー | パネル外のエリアはマウスクリックを透過しiRacingを操作可能 |

## 動作環境

- Windows 10 / 11（64ビット）
- iRacing がインストール済みであること（ライブテレメトリー使用時）
- .NET 8.0 ランタイム（インストーラー版は同梱のため不要）

---

## インストール

1. [Releases ページ](https://github.com/liquidfinger/iRacing_App_release/releases/latest) から `iRacingSlipAngleVisualizer_Setup_v1.2.8.exe` をダウンロード
2. インストールウィザードに従って完了
3. スタートメニュー または デスクトップショートカットから起動

---

## クイックスタート

1. iRacing を起動してセッションに参加
2. `iRacingSlipAngleVisualizer` を起動
3. コントロールパネルの **「Show Overlay」** ボタンをクリック
4. ゲーム画面にオーバーレイが表示されます

IBT ファイルを解析する場合は、コントロールパネルの **「Data Source」タブ** から IBT ファイルを選択して再生してください。
